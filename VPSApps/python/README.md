# VPSApps

Various apps I run on my VPS.

## Comics
Retrieve various comics and send them by email.

## Eventor
Send an reminder interval before some sort of event happens.

## Kwed
Send email with links to latest remix.kwed.org entries.

## Newsfeed
RSS to email feed.

## Lichess
Watch my Lichess games and send email when I am in turn.
