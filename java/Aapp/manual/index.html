Aapp Manual

Introduction

Aapp is a pre-processer running entirely as an Ant task. Pre-processing java files is generally frowned upon but sometimes a good pre-processor really can get the job done (especially in J2ME world :-)

Aapp tries to take the best from the rest and implement a couple of features of its own.

Aapp was written by Brian Schau (http://www.schau.com/).
