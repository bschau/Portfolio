Installation

Requirements

  • Ant (obviously ..) - I use ant-1.7.1.
  • Linux, Windows or Mac OS X. Aapp has been developen and used on Slackware Linux 12+, Windows Vista and Mac OS X 10.5 (leopard).

Installation

Binary

Copy Aapp.jar to f.ex. the Ant lib directory or some other directory of your choice.

Source

You need Ant to build Aapp. In the top-level directory of the source distribution type ant. When ant is finished copy src/Aapp.jar to f.ex. the Ant lib directory or some other directory of your choice.
