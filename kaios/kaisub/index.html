KaiSub

What?

What is KaiSub? KaiSub is a small KaiOS optimized web client for your Subsonic server.

Open this page in the webbrowser on your KaiOS phone to start the client.


Practical Info

  • The client only run in the webbrowser. From update v16 you can pin this site to the start screen for easy access.
  • No information is sent to the server where the KaiSub client is hosted. All credentials are stored on your phone and only sent to your Subsonic server.
  • The client is currently only limited to play podcasts.
  • The client is limited in functionality on purpose - all administration of podcasts should be done on the server.
  • The client support both http:// as well as https:// URLs. But you should really consider enabling https:// on your server.


Big Fat Warning!

This demonstration site runs on plain HTTP so all credentials are sent in plaintext. This is done to be compatible with the Subsonic Demo server. If you run this on your on premise feel free to configure this site and your Subsonic server to use HTTPS.
