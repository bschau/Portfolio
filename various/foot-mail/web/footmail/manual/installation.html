FootMail - Installation Manual

FootMail is a collection of php scripts. FootMail should be able to run on most apache+php or iis+php combinations if the following requirements are met:

  • A newer version of PHP is required, at least version 4.3.x. FootMail is developed and run on PHP-5.0.3.
  • PHP must support the imap library, at least version 2000.
  • Your webserver must be able to access your POP3 server.

FootMail is a single user installation. FootMail can run in a multiuser environment but it must be installed for every user.


Installation

Installation is simple:

  1. Transfer the FootMail.tar.gz archive to your Web server.
  2. Unpack the archive: 
    tar xjf FootMail.tar.gz
  3. You now have a directory on your webserver called FootMail-x.y. Open up a webbrowser and point to to 
    http://your.domain/FootMail-x.y/
    and click the Configure link to Configure FootMail.


Security

As you will have noticed by now, there is absolutely no built in security in FootMail. However, do not despair! You can enhance the security of the FootMail directory by:

  1. Apache: place a .htaccess file in the directory.
  2. IIS: add the FootMail directory as a web resource and place the appropriate permissions on the directory (and sub-directories/files).

If your client supports it, FootMail can be placed in a secure area of your webserver guarded by SSL or the like.
