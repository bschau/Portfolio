ltools
------


ltools is a collection of tools for developers.  The ltools suite consists of:

  lmake - a make replacement
  lcdoc - javadoc for c
  lhd   - a hex dumper
  lme   - a macro expander

Full documentation can be found at:

	http://www.schau.com/
