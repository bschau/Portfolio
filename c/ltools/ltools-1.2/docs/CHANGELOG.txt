2007-09-25	v1.2

	* (lme) Initial release.
	* (lmake) The return code from external commands are now handled
	  correctly.
	* (lmake) The OS system variable cannot be redefined.
	* (lmake) New '!' flag to suppress writing of the command to run.
	* (lmake) Fixed some path bugs.
	* (lcdoc) Relative paths are now handled correctly.

2007-09-08	v1.1

	* (lcdoc) Output is now HTML-4.01 strict compliant.
	* (lcdoc) Minor fixups to the man page.

2007-07-23	v1.0

	* This is the initial release of ltools. This release was based on:

		lmake v0.8
		lcdoc v0.1
		lhd v0.1

	* (lmake) Added the -h switch.
	* (lmake) Fixed a bug where $(something)$/else would not expand propely.
	* (lmake) Added the $0 variable expansion (all arguments to procedure).
	* (lmake) Added the basename command.
	* (lmake) Added the @ qualifer to force a certain command to look in
	  the filesystem.
