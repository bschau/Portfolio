HOW TO BUILD
============

Linux
-----

First:   chmod 755 build.sh

Run the build.sh script as root and the ltools will be built and installed.
Run the build.sh script as non-root and the ltools will be built.


Windows
-------

Run the build.bat script to build the ltools and, if you have Inno Setup
installed, you will also get a setup file.


Both
----

Go to http://www.schau.com/ for more information.
